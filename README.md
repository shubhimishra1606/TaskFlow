# TaskFlow

A high-performance, minimalist full-stack Task Management Dashboard built using the **MERN Stack** (MongoDB, Express, React, Node.js). Designed with a modern slate-teal premium interface, this application offers fluid tracking, smooth status pipeline manipulation, and seamless data synchronization without full-page reloads.


## ✨ Features

- **End-to-End CRUD:** Complete control over creating, fetching, updating, and purging sprint tasks.
- **Dynamic State Pipeline:** Tap any task status card to cycle its state natively through `Pending` ➡️ `In Progress` ➡️ `Completed`.
- **Granular Control Center:** Instant key-based search and filter matrices by **Priority Levels** (Low, Medium, High) and **Task Statuses**.
- **Robust Field Validation:** Front-end and back-end layer guards to intercept anomalous or empty node requests.
- **Premium Layout:** High-fidelity, minimal workspace theme.

## 🛠️ Tech Architecture

### Frontend
- **React.js** — High-speed hot module replacement framework.
- **Tailwind CSS** — Sleek utility-first structural layouts and design elements.
- **Lucide React** — Crisp, lightweight modern interface iconography.
- **React Hot Toast** — Non-blocking beautiful dashboard operational micro-logs.

### Backend
- **Node.js & Express** — High-throughput REST API controller network.
- **MongoDB & Mongoose** — Scalable cloud document persistence models.
